## Featured Quests
> Three projects that best represent what I build.

### 1) Nexus Sight — LoL Draft Win Predictor  
**Predictive draft analysis engine** that estimates win probability from draft + team strength.

- **Proof:** ~**71% validation accuracy** (XGBoost)  
- **Build:** FastAPI + React + TypeScript + SQLite  
- **Repo:** https://github.com/Zylocope/LOL-game-analyzer

### 2) EazyMove — Delivery Management System (DB + Web)  
A **Next.js delivery management** app built to showcase a real relational workflow: customers → orders → drivers → delivery state.

- **Proof:** full CRUD workflows + driver assignment + cost calculation (no GPS; formula-based)  
- **Build:** Next.js 14 (App Router) + TypeScript + Tailwind + shadcn/ui  
- **Repo:** https://github.com/Zylocope/EazyMoveApp

### 3) SEEiT-CV — Accessibility Computer Vision App  
A React Native app using computer vision + audio feedback to help visually impaired users navigate environments.

- **Proof:** real-time object detection + TTS + accessibility-first features  
- **Build:** React Native (Expo) + ML Kit + TensorFlow Lite  
- **Repo:** https://github.com/Kinosaur/SEEiT-CV
